# Child Incidents Dashboard

An interactive **Power BI dashboard** designed to analyze child incident data across countries, helping stakeholders identify trends, monitor geographical patterns, and support evidence-based decision making.

## Overview

This project transforms raw child incident data into an intuitive analytics dashboard that enables users to:

* Monitor overall child incident trends
* Compare incidents across countries and regions
* Explore geographical distribution of reported cases
* Identify changes over time
* Support safeguarding and programme planning through data-driven insights

## Dashboard Pages

### Executive Summary

A high-level overview of key indicators, allowing users to quickly understand the current state of child incidents.

### Geographical Analysis

Interactive maps and regional breakdowns that reveal where incidents are occurring and highlight geographical patterns.

### Trend by Country

Time-series analysis that enables users to compare reporting trends across different countries and monitor changes over time.
